# 血統くん — UI刷新プロトタイプ

漆黒×ゴールド refined・モバイル優先の UI 刷新デザイン。
**AI診断「3つの目」→ 総合判断** を中核に再構成しています。

## 見方
ローカルで簡易サーバを立てて `血統くん.html` を開いてください
（JSXを外部ファイルから読むため file:// 直開きでは動きません）。

```bash
cd prototype
python3 -m http.server 8000
# → http://localhost:8000/血統くん.html
```

ツールバーの **Tweaks** から、トーン4案（格式/疾走/ターフ/箔）・
AI診断の表現（カード/コンパクト/カルテ）・チャート様式（リッチ/ミニマル）を切替できます。

## ファイル構成（= App.jsx 分割の設計図）
| ファイル | 役割 |
|---|---|
| `血統くん.html` | エントリ。フォント・スクリプト読込 |
| `data.js` | データ層（宝塚記念18頭・AI診断・馬場傾向・1000回シミュ・カレンダー） |
| `theme.js` | トーン4案（CSS変数マップ） |
| `tweaks-panel.jsx` | Tweaksパネル（プロトタイプ専用・本番には不要） |
| `ui.jsx` | 共有UI（ヘッダー/ボトムナビ/枠番チップ/ゴールドバー/バッジ等） |
| `home.jsx` | 予想ホーム（NEXT RACEヒーロー＋重賞一覧） |
| `racedetail.jsx` | レース詳細シェル＋出馬表 |
| `diagnosis.jsx` | AI診断「3つの目」ナビ＋⚖総合判断 |
| `diagnosis-eyes.jsx` | ①血統 ②馬場傾向 ③1000回シミュ の各パネル |
| `analysis.jsx` | 分析チャート（距離適性/適性散布図/系統トレンド） |
| `extras.jsx` | 血統DB・馬券計算 |
| `app.jsx` | ルート（タブ/画面遷移/Tweaks配線） |

> このファイル粒度がそのまま、3,500行の `src/App.jsx` をコンポーネント分割する
> 際の指針になります（共有UI・画面別・3つの目を個別ファイルに）。

## 注意
- これは**デザインプロトタイプ**（ブラウザ内Babel）です。本番の React+Vite に
  落とし込む際は、Babel/CDN読込をやめて通常の import に置き換えてください。
- 配色・スコアロジックの表示値はモック用の調整値を含みます。
